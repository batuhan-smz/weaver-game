Drop your custom sound files into these folders with these exact names:

www/assets/sfx/place/place.mp3
www/assets/sfx/lineclear/lineclear.mp3
www/assets/sfx/clusterburst/clusterburst.mp3
www/assets/sfx/megaweaver/megaweaver.mp3
www/assets/sfx/clean/clean.mp3

Behavior:
- If the matching .mp3 file exists, the game plays that file.
- If it does not exist yet, the game uses the built-in synthesized fallback sound.

Suggested usage:
- place.mp3: stronger block placement hit
- lineclear.mp3: heavier row/column clear impact
- clusterburst.mp3: colorful burst / explosion sound
- megaweaver.mp3: biggest fanfare / powerful explosion chain
- clean.mp3: satisfying full-board clean reward sting
